非暴力沟通四要素

非暴力沟通(NVC)是一种有效的沟通模式，由马歇尔·卢森堡博士创立。它强调共情和诚实的表达，由四个核心要素组成。

点击下面的卡片，了解每个要素的详细内容和使用方法。

观察

客观地描述你所看到、听到或回忆的事实，不带评判、解释或推测。

什么是观察？

观察是指客观地描述发生了什么，就像摄像机记录的画面一样。观察是对事实的陈述，而非对事实的评价或解释。

观察的原则
  • 描述具体的行为和事件，而非泛泛的评价
  • 避免使用"总是"、"从不"等概括性词语
  • 区分观察与评价、解释或判断
  • 聚焦于当下或特定情境，而非过度概括
示例
✓ 观察

"昨天你迟到了30分钟。"

✓ 观察

"我看到桌子上有三个脏盘子。"

✗ 评判

"你总是迟到，太不靠谱了。"

✗ 评判

"你从不打扫，房间乱糟糟的。"

感受

表达你的情感体验和情绪状态，而不是想法、评价或解释。

什么是感受？

感受是指我们的情绪状态和身体感觉，它们是我们内在需要得到满足或未得到满足时的反应。感受与想法或评价不同，它们直接关联到我们的内在体验。

表达感受的原则
  • 使用具体的情感词汇，而非模糊的表达
  • 区分感受和想法（"我感到失望"vs"我觉得你不重视我"）
  • 承担对自己情感的责任，而非归咎于他人
  • 认识到感受源自我们的需要，而非他人的行为
常见的感受词汇

需要得到满足时：

  • 快乐、喜悦、满足
  • 平静、安心、放松
  • 感激、感动、温暖
  • 兴奋、充满活力、振奋
  • 充满希望、乐观、信任

需要未得到满足时：

  • 悲伤、失望、沮丧
  • 焦虑、担忧、不安
  • 愤怒、恼火、烦躁
  • 疲惫、无力、困惑
  • 孤独、怀念、思念
示例
✓ 感受

"我感到失望和沮丧。"

✓ 感受

"我现在很焦虑和担心。"

✗ 想法

"我觉得你不尊重我。"

✗ 想法

"我感觉被忽视了。"

需要

识别并表达引发你情感的内在需要、价值和渴望。

什么是需要？

需要是所有人共有的，对于生存和幸福至关重要的普遍价值和渴望。在非暴力沟通中，我们认为所有人的行为都是试图满足某种需要。需要与具体的行动策略不同，需要是普遍的，而策略是具体的。

表达需要的原则
  • 区分需要（普遍的）和策略（特定的满足方式）
  • 表达需要时不涉及特定人物或行为
  • 承认所有人都有相同的基本需要
  • 理解我们的感受源于我们的需要是否得到满足
常见的人类基本需要
  • 连接、理解、信任
  • 安全、保障、稳定
  • 尊重、认可、欣赏
  • 自主、选择、自由
  • 意义、目的、贡献
  • 诚实、真实、透明
  • 成长、学习、发展
  • 支持、合作、互助
  • 休息、放松、娱乐
  • 和谐、秩序、美
示例
✓ 需要

"我需要被理解和尊重。"

✓ 需要

"我渴望信任和安全感。"

✗ 策略

"我需要你给我发信息。"

✗ 策略

"我需要你改变态度。"

请求

明确、具体地表达你希望对方做什么，而非不要做什么。

什么是请求？

请求是我们为了满足需要而向他人提出的具体、积极、可行的行动期望。有效的请求清晰地表达我们希望对方做什么，而不是不要做什么。请求与要求不同，请求尊重对方说"不"的权利。

提出请求的原则
  • 使用具体、明确、肯定的语言（说明你想要什么，而非你不想要什么）
  • 请求应当是当下可以实现的具体行动
  • 区分请求和要求（请求接受对方说"不"）
  • 检查对方是否正确理解了你的请求
请求的类型
  • 行动请求：请求对方采取特定行动
  • 连接请求：请求对方表达他们的感受、想法或反应
  • 澄清请求：请求对方重复或解释他们所说的内容
  • 诚实请求：请求对方分享他们的真实感受和想法
示例
✓ 请求

"你能告诉我你现在的感受吗？"

✓ 请求

"我想知道你对这个方案的看法。"

✗ 要求

"你必须现在回复我。"

✗ 模糊

"你能不能重视一点？"
